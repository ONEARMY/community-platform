# Icon

Below you will be able to see how to use and configure a `Icon` component:

<Preview>
  <Story name="Default Icon">
    <Icon
      glyph={text('glyph', 'download')}
      size={text('size', 'md')}
      onClick={action('onClick')}
      color={text('color', '')}
    />
  </Story>
</Preview>

## Sizes

You can use one of the pre-determined size constants: `xs`, `sm`, `md`, `lg` and `xl` or simply set the prop `size` to an integer value.

<Preview>
  <Story name="Icon Sizes">
    {[
      <Icon glyph={'download'} size={'xl'} marginRight={'10px'} />,
      <Icon glyph={'download'} size={'lg'} marginRight={'10px'} />,
      <Icon glyph={'download'} size={'md'} marginRight={'10px'} />,
      <Icon glyph={'download'} size={'sm'} marginRight={'10px'} />,
      <Icon glyph={'download'} size={'xs'} marginRight={'10px'} />,
    ]}
  </Story>
</Preview>
